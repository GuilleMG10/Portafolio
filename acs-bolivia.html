PROJECT 01 · Medical · Non-Profit

ACS Bolivia Web Platform

Live in production Mar 2026 – Present acsbolivia.com ↗
Overview

Official web platform for the American College of Surgeons Bolivia chapter. Built for congress and student management, with a public-facing site receiving thousands of visitors and a private admin panel for the ACS board members.

Tech Stack
NestJSNext.jsPostgreSQLTypeScriptPrismaJWTRBACTailwind CSS
The Challenge

What problem needed solving

The ACS Bolivia chapter needed a unified digital presence that served two very different audiences at once: thousands of medical professionals visiting the public congress page, and a small group of board members managing registrations and content through a private admin panel.

The Solution

How I built it

Built a decoupled architecture with a public Next.js frontend and a secure NestJS backend. The admin panel uses role-based access control to restrict sensitive operations, while the public site is optimized for performance and SEO to handle high traffic during congress periods.

Key Features
Public Landing Page
High-traffic public site for congress announcements, speaker profiles, and event schedules. Optimized for thousands of concurrent visitors.
Private Admin Panel
Secure dashboard for ACS board members to manage registrations, content, and attendee data with full CRUD operations.
Registration System
Full registration flow with email validation, duplicate prevention, status tracking, and relational database design.
Direct Client Coordination
Worked directly with ACS board members through multiple requirement gathering sessions to ensure the platform matched their operational needs.
Results
✓ Thousands of medical professionals reached through the public platform
✓ Board members managing registrations independently through the admin panel
✓ Zero downtime since launch during congress periods
✓ Full audit trail for all registration and admin actions
