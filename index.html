SaaS · Backend Infrastructure

Multi-tenant SaaS API

Production-grade multi-tenant REST API designed to demonstrate Clean Architecture, SOLID principles, and the ability to build scalable SaaS infrastructure. Each tenant gets its own isolated PostgreSQL schema — the TenantInterceptor resolves the tenant from the X-Tenant-ID header on every request and scopes queries accordingly.

  • Schema-per-tenant isolation via PostgreSQL search_path
  • JWT auth with access + refresh token rotation
  • Stripe subscriptions, webhooks, and billing portal
  • Swagger/OpenAPI 3.0 docs · Docker + docker-compose
  • SUPER_ADMIN / ADMIN / MEMBER RBAC with custom guards
NestJS 10 TypeScript strict PostgreSQL 16 TypeORM Stripe API JWT Docker Jest Swagger
